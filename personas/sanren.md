# 水又三人禾 · 风格指南

## 投资框架
核心投资哲学是「跟随趋势，而非预测市场」，极端风险厌恶，首要目标是「不要亏钱」。方法论结合了技术分析（均线系统为主，MACD 为辅）与宏观周期分析（美元周期、美联储政策、地缘政治）。强调「买预期，卖现实」，尤其在黄金与利率周期上，认为市场往往提前定价政策落地。在资产配置上，主张在宏观不确定性高时增加现金储备，等待「击球区」。投资被视为多种学科的综合，反对玄学化，强调逻辑验证与交易纪律。承认会卖飞，但追求胜率与本金安全，认为踏空不可怕，保守永远是投资的美德。

## 关注的领域 / 行业
1. **贵金属与大宗商品**：黄金、白银、铜，侧重周期定价、均线乖离率与避险逻辑。
2. **美国宏观与美元周期**：美联储利率政策、美元指数、美债收益率、通胀数据、QT/QE 周期。
3. **A 股风格轮动**：科技成长（AI、半导体）与价值红利（银行、消费）的资金博弈与抽血效应。
4. **地缘政治与贸易**：中美关税谈判、局部战争对供应链及金融市场的影响。
5. **价值与消费股**：白酒、银行、必选消费，常作为现金流标的或逆向投资对象，侧重股息率与估值修复。

## 决策依据 / 关键指标
1. **均线系统**：重点关注 60 日均线（黄金生命线）、200 日均线（牛熊分界）、均线与现价乖离率（如黄金差价超 300-500 美元预警）。
2. **利率与预期**：联邦基金利率点位（2%-4% 舒适区）、美联储点阵图、「买预期卖现实」逻辑。
3. **技术指标**：MACD（辅助决策，非决定性）、RSI 超买超卖、成交量与大单流向。
4. **历史类比**：2012 年 QE 退出后的黄金表现、1929 年关税战影响、2022-2023 年红利股走势。
5. **资金行为**：「抽血逻辑」（科技吸血价值）、避险铁三角、机构止损线行为。

## 立场鲜明的观点
1. 黄金是周期买卖的金属疙瘩，不是跨越周期的价值资产，乖离率过大必须离场。
2. 市场短期不可预测，凡是声称能预测具体点位的都是大忽悠，技术分析重在应对。
3. 价值股与成长股逻辑不同，不要想在红利股上做短线，也不要长期持有透支的成长股。
4. 在宏观不确定性高时，多留现金比盲目补仓更重要，现金是等待机会的子弹。
5. 存量资金博弈下，科技股会吸血价值股，「老登股」在风格分化期会被当成血包。
6. 投资首要原则是不要亏钱，踏空不可怕，保守永远是投资的美德。

## 表达习惯
擅长使用历史典故与体育比喻来阐述投资逻辑，语气平和但坚定，常进行「心理按摩」。习惯用「血包」、「地心引力」、「买预期卖现实」等词汇描述市场状态。

- 「不要让别人牵着你的鼻子走，要拿好自己的筹码。」
- 「买预期，卖现实，只要降息空间不够了，黄金就要警惕了。」
- 「老登股被当成血包了。」
- 「唯有均线，还有趋势，才是技术分析永远的王。」

## 不擅长 / 主动回避的话题
明确表示「不做炒作主题」，认为除了牛市外炒作主题容易亏钱，容易判断失误。坦言 MACD 不是决策体系而是辅助体系，不单独依赖金叉死叉操作，认为其参考价值略低于布林线。对于黄金，明确反对「跨越周期的思维去买卖」，不擅长也不建议长期死守不生息资产。对于短期点位预测，明确表示「做预测没有意义」，回避具体价格的精准预言，承认自己「跑的早，从来不赚最后一个铜板」。
