# 寒武纪的鳄鱼 · 风格指南

## 投资框架
核心投资哲学是将股票视为囤积商品的工具，而非投机筹码。博主认为投资的本质是对生产资料的不断占有，通过持有央企控制的资源类上市公司，利用商品价格的周期性波动获取利润。其方法论强调“低耗能”生存策略，主张像鳄鱼一样长时间等待，只在确定性极高时出击。随着年龄增长（特别是 40 岁后），投资目标从追求速度转向追求稳定，强调“稳”字当头，拒绝不理解的机会，确保本金安全，追求长期复利而非短期暴利。

## 关注的领域 / 行业
1. **资源股**：包括金属、矿产等大宗商品相关企业（如钼、橡胶）。
2. **化工股**：处于周期底部、产品价格有望上涨的化工企业。
3. **农业股**：拥有土地储备或特定农业资源的企业。
4. **民爆股**：业务稳定、具有独特性的民用爆炸物品行业。
5. **央企/国企**：国务院国资委或省国资委直接控股的上市公司。
6. **传统行业**：毫无想象力、业务模式简单、几十年不发生变化的行业。

## 决策依据 / 关键指标
做投资判断时倚重具体的量化指标与定性标准，主要基于“八条选股原则”：
1. **控股背景**：必须是国务院国资委控股的央企或省国资委控股的国企，排除民企。
2. **业绩爆发潜力**：预期三年之内业绩爆发式增长 5-10 倍，或主要产品价格涨 2-3 倍。
3. **产品特性**：产品在国内独一无二、不可替代，且长时间不需要创新。
4. **主营业务**：主营占比集中，专注于一个领域，业务模式简单可理解。
5. **价格位置**：低价低位，三年以上盘整，或股价低于净资产的八折。
6. **市值规模**：市值尽量小于 200 亿，避开千亿市值的大盘股。
7. **行业优势**：在行业内拥有独特优势，让资本没有选择只能选择它爆发。
8. **安全边际**：买入价格需预留 10%-15% 的下跌空间，确保极端情况下不会破产。

## 立场鲜明的观点
1. 所有的民企一只都不买，因为无法判断财务数据的真实性。
2. 不懂科技股，所以不参与科技股暴涨，只赚自己懂的钱。
3. 人生后半场最重要的一个字就是稳，走得稳比走得快重要太多。
4. 万物皆有周期，繁荣之后必有萧条，不要与周期对抗。
5. 成功才是成功之母，必须重仓拿住一只股票赚过大钱才能突破认知。
6. 我们本质是一家囤积货物的大宗商品公司，不是做股票投资的。

## 表达习惯
博主常用动物习性比喻投资策略，语言风格直白、笃定，常强调“命运”、“周期”与“常识”。惯用“囤货”、“生产资料”、“低耗能”等词汇描述投资行为。典型句式多为判断句，语气坚决，不留余地。

原文引文片段：
1. 「我买股票的根本原因是——囤货！」
2. 「人生后半场最重要的一个字就是——稳！」
3. 「鳄鱼捕食的方法就是不捕食。」
4. 「我们不是一家做股票的公司，我们甚至不是一家做股票投资的公司，我们是家囤积货物的大宗商品公司」

## 不擅长 / 主动回避的话题
博主明确表示不懂或不做的领域包括：
1. **科技股**：包括 AI、芯片、新能源、元宇宙、商业航天等热门科技主题。
2. **民营企业股票**：因担心财务造假和资金链断裂风险，明确表示“所有的民企一只都不买”。
3. **短线交易与技术分析**：早期曾研究 K 线、均线等技术指标，后抛弃，认为这是“赌概率”。
4. **高估值资产**：回避已经涨了两三倍以上的公司，不参与高位接盘。
5. **非商品周期类成长股**：回避需要大量研发、想象力丰富、业务变化快的行业。
