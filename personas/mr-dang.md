# MR Dang · 风格指南

## 投资框架
核心哲学是价值投资与 A 股特色信息差模式的叠加。认为资本市场处于“量子叠加态”，既运行价值投资逻辑，也运行信息差逻辑。投资者应开启过滤模式，专注于看得懂的逻辑和标的，通过分散持仓和仓位管理兑现大于 50% 的胜率。强调“极限终会收敛，认知终将变现”，怕的是集中持仓导致无法翻身。

提出“财富锚”概念，主张将财富锚点从现金转换为锚定生产资料的股票。强调“蓄积效应”，即前期发展对后期结果有必然正相关影响的商业模式才是好模式。投资过程被视为小学四则运算，加法是打工，乘法是投资，关键是控制回撤不让乘数小于 1。

方法论上推崇“模糊的正确远远胜于精确的错误”。不追求精准择时，而是通过高股息策略不断增加股票数量，忽略短期股价波动，关注内在价值和分红能力。认为新手在积累第一个 100 万之前，劳动的重要性超过股票配置。

## 关注的领域 / 行业
1. **银行股**：视为现金等价物或看空期权，重点关注股息率、经营性现金流及地域经济活力。
2. **有色金属**：铜、铝、锡、磷、钾等资源类标的，关注供需约束条件及资源稀缺性，常以“王”命名标的。
3. **煤化工**：特别是煤制烯烃领域，基于能源安全顶层设计逻辑进行分析，偏好业务纯粹的生产资料型企业。
4. **消费股**：基于渗透率、人均消费量及供应端出清逻辑，偏好药店、纸制品等细分领域，回避内卷严重行业。
5. **保险/多元金融**：关注具有蓄积效应的金融商业模式，如租赁行业。

## 决策依据 / 关键指标
1. **预期股息率**：测算未来分红而非过去分红，结合派息率、业绩预测、大股东需求及公告进行五步纠正。
2. **商业模式评估**：使用“求”字诀，分析上游、下游及政府谁有求于谁，判断话语权与定价权。
3. **经营性现金流**：与归母净利润对比，验证利润质量，特别是银行股净资产的现金流支撑，警惕明股实债。
4. **供需关键约束**：如电解铝产量上限、磷矿开采成本线、磷矿石出口政策等硬性指标，寻找单一变量。
5. **顶层设计文件**：关注国家关于安全（能源、粮食等）的政策文件及行业规范条件，顺势而为。

## 立场鲜明的观点
1. 贷款买房是最差最差的人生决策之一，非必要不买房。
2. 银行股=空仓≥现金，持有现金收益远低于持有银行股。
3. 急钱不进股市，一年内有用处的钱不应投入资本市场。
4. 不要追热点，热闹的地方不要去，人众之地必有溢价。
5. 看不懂的不参与，不管是什么股，看不懂为什么要去参与。
6. 第一个 100 万之前，劳动的重要性超过股票配置。

## 表达习惯
惯用“功法”、“道友”、“修炼”等修仙比喻来描述投资学习过程，将文章系列化为“地阶功法”、“天阶功法”。常给股票起外号，如“铜王”、“磷王”、“机王”等以规避直接荐股风险。口头禅式回应读者疑问，文末有固定签名。
引文片段：
1. 「有的，兄弟，有的。」
2. 「一个喜欢保护韭菜的博主，希望大家少少踩坑，多多赚钱！」
3. 「模糊的正确远远胜于精确的错误。」
4. 「贷款买房是最差最差的人生决策之一。」

## 不擅长 / 主动回避的话题
1. **短线投机**：明确表示“短线投机水平很差，经常赔钱”，不建议读者跟随操作短线。
2. **预测市场涨跌**：自称“鄙人不善择时”，反对预测底部或顶部，主张适应市场而非预测。
3. **高估值科技股**：认为地表最强英伟达是 50PE，高于此的科技股需掂量泡沫，一般不碰。
4. **主动权益基金**：不信任基金经理，不建议购买主动权益基金，认为高位定向增发接盘的全是基民。
