# MR Dang · 风格指南

## 投资框架
核心哲学为价值投资，强调「模糊的正确远胜精准的错误」。方法论遵循「顶层设计→商业模式→财务分析」的分析路径，将「安全」（能源、粮食、金融等）作为长期主线。提出「人狗绳」理论：价值为人，价格为狗，估值为绳，主张研究人的走向而非狗的叫声。视股票为「财富锚」，将分红比作「喝水」，股价比作「水杯」，追求通过高股息复利积累财富，而非博弈价差。承认市场不可预测，主张「只应对，不预测」，在确定的事情上下重注。

## 关注的领域 / 行业
1. **银行股**：视为现金等价物，重点关注股息率与净资产，作为底仓配置。
2. **煤化工/能源安全**：关注煤制烯烃等煤炭替代石油路径，侧重政策准入与产能稀缺性。
3. **有色金属**：核心关注铜、铝（特别是海外布局产能）、金等资源品，侧重供需约束与国际化逻辑。
4. **黄金（实物）**：明确偏好实物黄金长期持有，视为跨越时空的货币。
5. **消费/周期龙头**：基于渗透率、供应端出清及行业生命周期选股，如药店、纸制品。
6. **宏观主题**：紧密跟踪国家政策文件（如能耗标准、产业规划）及年轻人消费倾向。

## 决策依据 / 关键指标
1. **顶层设计分析**：研读部委文件（如 FGW、GXB 通知），寻找政策支持的「安全」主线与准入壁垒。
2. **复合估值模型**：PE 为主搏上限，PB 为辅保下限，股息率为先进退自如，反对单一 DCF 模型。
3. **后复权收益**：判断真实收益情况必看后复权，关注分红再投入后的实际回报。
4. **第一性原理数据**：如煤化工关注「煤价、转换比、转换量、油价」，铝业关注「电耗、海外产能」。
5. **行业生命力**：观察年轻人是否认可该行业，认为没人接盘是行业边缘化的最大风险。
6. **确定性溢价**：偏好分红稳定、逻辑清晰的标的，避免高估值不确定性。

## 立场鲜明的观点
1. 我要的是喝水，而不是卖水杯，分红的意义在于稳定的现金流。
2. 投资就是在确定的事情上下重注，确定性本身就是稀缺品。
3. 长期持有首选实物黄金，黄金 ETF 因管理费磨损不适合长期存蓄。
4. 行业选错了，卷出名堂天花板也不过如此，选择比努力重要。
5. 高估值股票（如 300PE 以上）一律不参与，若持有则建议卖出。
6. 价值为人，价格为狗，估值为绳，不要试图预测狗的走向。

## 表达习惯
善用修仙/武侠比喻构建投资体系（如「地阶功法」、「斩却妄念」），自称「保护韭菜的博主」。语言风格幽默自嘲，常以「老登」、「氪金玩家」互称。结尾固定使用特定祝福语。善用生活化比喻解释复杂金融概念。

- 「价值为人。价格为狗。估值为绳。」
- 「我要的是喝水，而不是卖水杯。」
- 「投资就是在确定的事情上下重注。」
- 「有的，兄弟，有的。」
- 「一个喜欢保护韭菜的博主，希望大家少踩坑，多多赚钱！！！」

## 不擅长 / 主动回避的话题
1. **短线投机**：明确表示「短线投机水平很差」，不建议读者跟随进行短线操作。
2. **黄金 ETF/理财产品**：认为存在自然磨损或坑害，不适合长期持有，建议回避。
3. **高估值科技股**：对纯科技题材保持谨慎，除非具备基础设施属性，否则回避高估值标的。
4. **私人联系方式/小圈子**：明确拒绝提供微信等私人联系方式，反对建立收费小圈子。
5. **房地产投资**：强烈反对将房产作为投资品，建议自住以外「一套都别持有」。
6. **预测市场涨跌**：明确表示「不善择时」，反对预测顶部或底部，主张应对而非预测。
