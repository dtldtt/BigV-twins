# 已清仓标的复盘 Prompt A/B 对比报告

**日期**: 2026-06-10  
**测试数据**: 飞龙股份 002536（建仓→补仓→清仓，+32.7%，137 天）  
**模型**: Qoder ultimate

## 两个版本的设计理念

| | 版本 A（结构化报告） | 版本 B（对话式教练） |
|---|---|---|

## 质量对比

### 版本 A 的优点

1. **3 条可复用规则**，每条有「证据 + 应用」结构，直接可以存入投资规则知识库
2. **纪律执行**单独一段，对"计划兑现度"做了系统评价
3. **"如果重来"**那段发现了用户没意识到的问题 — "仓位规划要更前置，把补仓预算在心理上提前花掉"
4. 结构完整，适合归档回看

### 版本 B 的优点

1. **"差点没扛住"的深挖** — "你的真实止损线是情绪崩溃的那个点，而不是计划里写的数字" — 这句话比 A 的整段"纪律执行"更有冲击力
2. **"纪律执行靠的不是意志力，是让自己处于不需要意志力的位置"** — 这种洞察 A 里没有
3. 更短更有力，读起来不像报告，像真人在跟你聊
4. "这句话比这笔赚的 2810 块值钱得多" — 有温度

### 各自的不足

**A 的不足**：
- 6 段结构有点长，"纪律执行"和"过程评价"有重叠
- 语气偏冷，像机构的合规报告

**B 的不足**：
- 只提炼了 1 条规则（A 有 3 条）
- 没有明确的"逻辑验证"环节（赚的是 alpha 还是 beta 有提到但没深入）
- 对于归档回看不够结构化

---

## 我的分析和建议

**推荐：A 的结构 + B 的语气**

A 的规则提取和结构化归档更适合长期积累（和趋势追踪的投资规则知识库对接），B 的洞察力和表达力更能触动用户。

最佳方案是**融合版**：
- 用 B 的语气风格（对话式、有温度、直接）
- 保留 A 的核心结构（但精简：不需要 6 段，合并为 4 段）：
  1. 赚亏真相（含 alpha/beta 拆解）
  2. 做对了什么 + 问题在哪（合并，不重复）
  3. 教训提取（2-3 条可复用规则，带证据和应用）
  4. 如果重来 + 一条最重要的话
- 字数：600-1000 字
- 语气：像 B 那样直接，不像 A 那样冷

## 版本 A 完整输出

### 一、交易总结
2025-04-28 建仓飞龙股份，6 月中旬下跌 18% 后按计划补仓摊薄成本至约 ¥17.18，9 月触及目标价 ¥22.80 全部清仓。持有 137 天，盈利 +32.7%，年化约 87%，**显著跑赢同期沪深 300（+5.2%）约 27 个百分点**，alpha 明显。

### 二、逻辑验证
买入逻辑（汽车热管理龙头 + 新能源渗透率提升）被市场验证：8 月起汽车零部件板块整体反弹，飞龙跟随并走出独立行情。**这次盈利偏 alpha 而非 beta**——同期大盘仅 +5%，板块贡献了一部分 beta，但 32.7% 的收益主要来自个股选择和补仓时机，逻辑对得起回报。

### 三、过程评价
- 建仓仓位（¥5,550）相对保守，给后续补仓留了空间，结构合理。
- 加仓严格按"跌 15% 补一笔"的预案执行，未提前抄底也未恐慌减仓，**计划兑现度高**。
- 6 月加仓后随笔写"仓位有点重，希望下半年能回来"，说明情绪上承压，但行为上没动作——这是值得肯定的"知行分离"。

### 四、纪律执行
- 止盈：触及 +30% 目标后**当日清仓**，干净利落。
- 止损：未触发（最大回撤约 -18%，离 -25% 止损线尚有距离）。
- 计划兑现度接近 100%。

### 五、教训提取

**规则 1：基本面未变 + 未触发止损 = 按预案补仓，不因情绪改剧本**
- 证据：6 月跌至 ¥15.20 时心理压力大，但仍按计划执行，摊薄成本贡献了约一半利润。
- 应用：持仓回撤 15-20% 时，先核对基本面逻辑，成立则按原计划补仓。

**规则 2：达到目标价就执行，不为"还能涨多少"重新谈判**
- 证据：清仓后心里盘算"拿到 25 能多赚"，这是事后偏差。
- 应用：目标价是"风险预算用完的信号"，触发即执行。

**规则 3：建仓 60% + 预留补仓子弹 40%**
- 证据：首仓 ¥5,550、补仓 ¥3,040，资金分布让你扛住了 18% 回撤。
- 应用：中长期标的默认"分两批进场"。

### 六、如果重来一次
你没意识到的是：补仓时的心理压力反映仓位规划还可以更前置——建仓时就明确写下"准备用 ¥3,000 补仓，这笔钱已经心理上花掉了"。

## 版本 B 完整输出

**赚亏拆解**

这笔 +32.7%，同期沪深 300 才涨 5.2%，超额收益 27 个百分点。拆开看：选股贡献了大头——你选对了赛道和时机。加仓拉低了成本均价从 18.5 降到 17.18，这一步直接把收益率从 25% 拉到 33%。运气成分不大，8 月板块反弹是行业基本面兑现。这笔赚的钱，大部分是认知变现，不是撞上的。

**过程中做对的事**

两个动作值得记住：第一，15.2 补仓。计划写了"跌 15% 补"，实际跌了 17.8% 你执行了。第二，22.8 清仓。目标 +30%，实际到了 +32.7% 你走了，没贪 25 块。你自己说"纪律就是纪律"——这句话比这笔赚的 2810 块值钱得多。

**真正的问题**

补仓时"差点没扛住"，补完后"仓位有点重"。这说明你的仓位规划有 bug——你的计划是理性的，但你的仓位大小没有匹配你的心理承受力。

"差点没扛住"这个时刻最值得挖：如果当时多跌 5%，你会不会割？如果会，说明你的止损线（-25%）其实是假的——你的真实止损线是情绪崩溃的那个点，而不是计划里写的那个数字。

**如果重来**

仓位分配可以更聪明：初始建仓减到 200 股，补仓加到 300 股。总投入差不多，但浮亏压力小很多。纪律执行靠的不是意志力，是让自己处于不需要意志力的位置。

**一条规则**

建仓时，先算"如果触发补仓，我的总仓位是多少"，确保补仓后仍在你睡得着觉的范围内。如果补完会让你焦虑，说明初始仓位开大了——砍掉 1/3 再开。
